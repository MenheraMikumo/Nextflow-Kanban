# Nextflow-Kanban
A simple Nextflow weblogger

## Usage

run nextflow (version>=0.31) with `-with-weblog {your server url}:{port}/listen/` or set them in nextflow.config.
```
weblog.enabled = true
weblog.url = "{your server url}:{port}/listen/"
```
